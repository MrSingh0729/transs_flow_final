‚ö° 4Ô∏è‚É£ How to show this immediately after scanning

You can use a QR scanner + fetch request setup like this (for Django / JS frontend):

<video id="preview"></video>
<div id="data"></div>

<script src="https://unpkg.com/html5-qrcode"></script>
<script>
  function onScanSuccess(qrCodeMessage) {
    fetch(`/api/fetch_lark_record/?url=${encodeURIComponent(qrCodeMessage)}`)
      .then(response => response.json())
      .then(data => {
        document.getElementById("data").innerHTML = `
          <b>Employee ID:</b> ${data.employee_id}<br>
          <b>Name:</b> ${data.name}<br>
          <b>Group:</b> ${data.group}<br>
          <b>Department:</b> ${data.department}<br>
          <b>CTQ Stage:</b> ${data.ctq_stage}<br>
          <b>Total Marks:</b> ${data.total_marks}
        `;
      });
  }

  const html5QrCode = new Html5Qrcode("preview");
  html5QrCode.start({ facingMode: "environment" }, { fps: 10, qrbox: 250 }, onScanSuccess);
</script>


Then your backend (/api/fetch_lark_record/) will:

Extract record_id from the QR link

Call Lark API using the record_id

Return only relevant fields to frontend.
